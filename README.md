# Simulator
Simulation of Dynamic Systems
This Program simulates Dynamic Systems
like Logistic Growth, Tentmap, Parabolic Iteration
Elliptic Billard, Stadium Billard, Oval Billard
It is written in VB.NET using the Community Version 2022
of MS Visual Studio.

All documentation is in the bin/Debug folder (pfd-files)
